# STI Dashboard — 38 Years, Annotated

Interactive dashboard for the Straits Times Index and its 30 constituents:
daily closes back to **December 1987** for the index (2000s onward for most
constituents), overlaid with 46 curated market events — crises, Singapore stories, policy
shifts and index milestones — plus every earnings release Yahoo records for
each constituent, annotated with whether it beat consensus EPS.

**Live:** enable GitHub Pages on this repo (Settings → Pages → deploy from
`main`, root) and it serves as-is. No build step, no dependencies.

## Features

- Canvas line chart with crosshair tooltip, wheel-zoom, drag-pan and an
  overview brush strip
- Clickable event flags on the chart + event cards below; selecting one zooms
  to ±18 months around it
- Ticker switcher for the index + 30 constituents, grouped by sector, showing
  each ticker's return over the currently shown window
- Range presets (1W / 1M / 3M / 1Y / 5Y / Max — opens on 1M), linear/log scale, event-category filters (incl. earnings)
- Light/dark theme (follows system, manual toggle), yearly data table

## Files

| File | Purpose |
|---|---|
| `index.html` | The whole app — markup, styles and chart code, self-contained |
| `data.js` | Price history (`window.STI_DATA`), generated by the fetch script |
| `earnings.js` | Earnings dates + EPS vs consensus per constituent |
| `logos.js` | Company favicons embedded as data URIs |
| `fetch_data.py` | Refreshes `data.js` from Yahoo Finance (`pip install yfinance`) |
| `fetch_earnings.py` | Refreshes `earnings.js` |
| `fetch_logos.py` | Refreshes `logos.js` |

## Updating the data

```bash
python3 fetch_data.py       # prices
python3 fetch_earnings.py   # earnings history
python3 fetch_logos.py      # logos (rarely needed)
```

Events are curated by hand in the `EVENTS` array at the top of the script in
`index.html`.

## Notes

- Prices are unadjusted daily closes from Yahoo Finance; long-history series
  reflect Yahoo's records, including any splits/re-listings as Yahoo reports them.
- Earnings history depth varies by ticker (Yahoo's records); a few tickers
  (Hongkong Land, CapitaLand Investment, MPACT, FCT) have none.
- Not investment advice.
